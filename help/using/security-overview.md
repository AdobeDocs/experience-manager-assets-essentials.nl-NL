# Beveiligingsoverzicht voor Assets Essentials {#security-overview}

De proactieve benadering van de veiligheid en de strenge procedures van Assets Essential helpen de veiligheid van de oplossing van Assets Essentials en klant vertrouwelijke gegevens beschermen. De veiligheid van digitale ervaring wordt voortdurend gecontroleerd om voor kwaadwillige activiteiten te blijven en de hulp verzekert de veiligheid van de gegevens van de klant.

Verwijs naar [ het Witboek van de Veiligheid van Assets Essentials ](https://www.adobe.com/content/dam/cc/en/trust-center/ungated/whitepapers/experience-cloud/adobe-experience-manager-assets-essentials-security-overview.pdf) voor een diepgaand veiligheidsoverzicht.
